# Quiet Diary Design System

Use a private, gentle, quiet visual language. The diary should feel like a locked personal book rather than a social feed.

## Tokens

- Ink: `#28231f`
- Linen: `#f6efe5`
- Paper: `#fffaf2`
- Surface: `#fffdf8`
- Moss: `#6f8067`
- Rose: `#b86f71`
- Clay: `#a65f45`
- Night: `#30343f`
- Radius: `8px`
- Heading font: Palatino-style serif
- UI font: Trebuchet/Verdana

## Components

- Locked cover: book-shaped rectangle, subtle internal border, no decorative icon clutter.
- Entry card: compact, one-level card only, with date, mood, optional text, tags, and media grid.
- Theme swatch: cover preview plus name and icon style.
- Touch target: 44px minimum.
